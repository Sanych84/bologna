Html & Css - основа любого сайта

Html и Css – это не языки программирования, тем не менее обе эти технологии являются основополагающими в процессе создания сайтов. Html – это язык разметки web-страницы, Css – язык оформления внешнего вида web-страницы, также называемый каскадными таблицами стилей. Две эти технологии являются взаимодополняющими друг друга и изучаются одновременно.
